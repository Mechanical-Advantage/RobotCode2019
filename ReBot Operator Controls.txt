ReBot Operator Controls

Intake
-retract hatch intake = A
-extend hatch intake = X
-intake cargo = B
-eject cargo = Y
-raise intake = right bumper
-lower intake = left bumper

Elevator
*All set points based on last intake operated*
-move to floor = POV up	
-move to ship = POV right
-move to rocket L1 = POV down
-move to rocket L2 = POV left
-force hatch set points = back
-force cargo set points = start
-manual drive = left stick

Climber
-manual drive = right stick
-enable vacuum = right trigger
-disable vacuum = left trigger
